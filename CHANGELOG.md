# Changelog

All notable changes to the **UZH Graphic Studio** project will be documented in this file.

## [1.3.0] - 2026-06-03

### Added
- **Independent Social Image Adjustments**: Zoom, horizontal position (OffsetX), vertical position (OffsetY), and brand gradient overlay settings are now completely independent for Print & Digital layouts versus Social Media layouts.
- **Form UI Separation**: Split the Image Control Panel into distinct "Print & Digital Adjustments" and "Social Adjustments" sub-sections for clearer editing.
- **Optional Image Credits**: Ensured that the image credit fields for both Print & Digital and Social category formats are completely independent and optional.

---

## [1.2.0] - 2026-06-03

### Added
- **Multi-Format Export Settings**: Print layouts (A0, A3, A4, A5) now automatically export to high-resolution PDF formats, while Social Media (Square, Portrait, Story) and Digital formats export as PNG and JPEG image assets.
- **Independent Social Text Fields**: The Headline title and font size multipliers are now separate for Print & Digital vs. Social layouts.
- **Custom Brand Gradient Overlay Presets**: Implemented dynamic swatches mapping to the active color palette's 4 colors.
- **Clean Social Layout Templates**: Hidden faculty headers, UZH logos, and details boxes on Social templates, ensuring full-bleed background images cover the canvas area edge-to-edge.

---

## [1.1.0] - 2026-05-29

### Added
- **AI-assisted campaign copy refinement** panel with sessionStorage caching.
- **QR Code generation** for print layouts.
- **Unsplash API integration** for live image search.
- **Color swatch support** based on UZH branding styles.

---

## [1.0.0] - 2026-05-27

### Added
- **Initial release** containing core layout engine, templates (A4, Screen, Social), drag-and-drop elements, and local export compilation.
